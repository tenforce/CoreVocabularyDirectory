Browse the metadata registry

Using the ontology inspired by the ISO 11179-3 meta-model, the following libraries have been modelled:

  • Core Vocabularies,
  • DCAT/ODS Mappings:
    • Browsable View,
    • Tree Visualisation of the DCAT/ODS mapping
    • EuroVoc Value Mapping Table
  • OASIS UBL 2.1 Common Library,

Examples of some registry entry points are the following examples (Core Vocabularies):

  • the Person Familty Name property (Core Vocabularies);
  • the Address Post Code Property;
  • the Person Country Of Birth Property;
